# AfVro TV - IPTV Playlist Indonesia

Koleksi channel IPTV Indonesia yang sudah diuji dan dapat digunakan, dilengkapi dengan icon channel yang diperbarui berdasarkan logo resmi dari berbagai sumber.

Sumber yang digunakan dalam playlist ini berasal dari berbagai repository dan website seperti:
- [iptv-org repository on Github](https://github.com/iptv-org/iptv)
- [IPTV Cat](https://iptvcat.com/indonesia__8)
- Live stream gratis dari website resmi seperti [Trans TV Live](https://www.transtv.co.id/live), [Klik TVRI](https://klik.tvri.go.id/), [RodjaTV Live](https://rodja.tv/), [RRI NET](https://rri.co.id/stream/video), [NusantaraTV Live](https://nusantaratv.com/live), [DENS.TV](https://www.dens.tv), dan lainnya.

Semua link dapat digunakan di [KODI](https://kodi.tv/download/), [OTT Navigator](https://ottnav.github.io/faq.html#ott-navigator-app-availability), TiviMate, dan player IPTV lain yang mendukung file m3u.

# Cara Setup
## Kodi
1. Install **IPTV Simple Client Add-on** (tersedia di **Add-ons > PVR clients**)
2. Install **InputStream Helper Add-on** (tersedia di **Add-ons > Program add-ons**)
3. Konfigurasi **InputStream Helper** > **(Re)install Widevine CDM library...** > Terima EULA, pilih versi terbaru, lalu tunggu hingga proses download dan instalasi selesai
4. Konfigurasi **IPTV Simple Client** > **Add add-on configuration** > **General** > **Location: Remote path (Internet address)** > Masukkan link playlist utama di atas ke **M3U playlist URL** > Klik **OK** untuk menyimpan konfigurasi

## OTT Navigator
- **Settings** > **Provider** > **ADD PROVIDER** > **Playlist** > Masukkan link playlist utama di atas ke **URL address** > **APPLY** > Set **Active** > **APPLY**

## VLC Player
1. Buka VLC > **Media** > **Open Network Stream**
2. Paste link playlist di atas, atau
3. **File** > **Open File** > pilih file `playlist.m3u`

## Android (OTT Navigator / TiviMate / IPTV Smarters)
1. Install aplikasi dari Play Store
2. Pilih opsi **Playlist** atau **M3U**
3. Masukkan link TiviMate di atas (recommended untuk TiviMate)
4. Masukkan link playlist dari GitHub
